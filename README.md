# User Management System (MVC)

A user management system built with TypeScript, Node.js, MongoDB, and EJS following the MVC pattern.

## Features
- User registration and login
- Password encryption
- Session management
- CRUD operations
- MVC architecture

## Tech Stack
- TypeScript
- Node.js (Express)
- MongoDB (Mongoose)
- EJS
- CSS

## Setup
1. Clone the repository
2. Run `npm install`
3. Add your `.env` file
4. Run `npm run dev`
